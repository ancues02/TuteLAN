# TuteLAN

TuteLAN

Práctica final de Redes y Videojuegos en Red realizada por Andrés de la Cuesta y Pablo Rodríguez-Bobada.

La memoria de la práctica la ha entregado Andrés.

Hay un Makefile que compila todo correctamente (es necesario tener instalado SDL, SDL_Image y SDL_TTF).

Para ejecutar el servidor hay que escribir en la consola ./ts ip puerto. Un ejemplo sería ./ts 0.0.0.0 2000.
Para ejecutar un cliente hay que escribir en la consola ./tc ip puerto nick. Un jemplo sería ./tc 127.0.0.1 2000 Pepe

El input está explicado en la memoria pero se hace a través de teclado.

Para poder jugar es necesario conectar 4 clientes a la vez.
